About Shopfinity

Shopfinity is a fully functional prototype food e-commerce platform designed specifically for testing review verification browser extensions.

All products, prices, orders, and user accounts are simulated. No real transactions are processed.

Built with plain HTML, CSS, and JavaScript — no frameworks, no backend, no database.
